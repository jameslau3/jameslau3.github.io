Book Database

Description: A database that stores information about books. It basic functionality using SQL queries provided, which include deleting, displaying, adding, and updating a book. This was a special project coded in two separate groups where we created a back end for the first sprint, and then required to create a front end for another group for the second sprint. The front end is built using Typescript, React, and JavaScript used with Next.js to connect to a server and generate the web application and display. The back end is built using PHP.
Skills: TypeScript, JavaScript, React, Node.js, Next.js, Database Query, PostgreSQL, Git/Github, VS Code, HTML, CSS, Docker, API Documentation
Team Size: Full Stack Developer in 4 Person Team, paired with another group.
